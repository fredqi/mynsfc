# mynsfc

A CTeX-based template for writing the main text of NSFC proposals, aligned with the 2026 General Program specification.

Version 2.00 adds `\nsfcpart` and `\nsfcsection` to automatically generate the official section headings of the 2026 General Program format. Although it defaults to the 2026 General Program structure, the template is designed to be flexible and extensible, allowing applicants for other programs (e.g., Young Scientists Fund, Key Program) to conveniently adapt section titles to their submission requirements.
